Student Performance Analysis Using Machine Learning

Project Overview

This project was developed as part of the Data Science course at the University of Engineering and Technology (UET) Peshawar. The objective of this project is to analyze student performance data and build machine learning models to predict students' final grades based on demographic, academic, family, and social factors.

The project follows the complete Data Science pipeline, including data preprocessing, exploratory data analysis (EDA), feature engineering, feature selection, model development, model evaluation, and business insight generation.

Project Objectives

• Analyze factors affecting student academic performance.
• Perform data cleaning and preprocessing.
• Conduct Exploratory Data Analysis (EDA).
• Apply feature engineering and feature selection techniques.
• Build and evaluate machine learning models.
• Compare model performance using different evaluation metrics.
• Generate meaningful insights and recommendations.

How to Run

1. Download or clone this repository.
2. Install the required Python libraries:

pip install pandas numpy matplotlib seaborn scikit-learn

3. Open the Jupyter Notebook file:
student_performance_analysis.ipynb

5. Run all cells from top to bottom.
